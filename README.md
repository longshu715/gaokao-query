# 高考志愿「四位一体」联动查询系统

## 概述

面向高考志愿填报师及学生的专业-大学-城市-产业深度匹配查询工具，支持四种反向检索与穿透对齐能力。

## 核心功能

| 查询入口 | 查询链路 |
|---------|---------|
| 📚 **按专业查** | 专业 → 相近专业群 → 优势大学(分层) → 就业城市 → 产业图谱 |
| 🏫 **按高校查** | 高校 → 优势专业 → 就业集聚城市 → 支撑产业 |
| 🏙️ **按城市查** | 城市 → 核心产业(纲目) → 特色大学 → 对口专业清单 |
| 🏭 **按产业查** | 产业 → 产业链聚集城市 → 产学研核心大学 → 精准匹配专业 |

## 数据维度 (2026标准)

- **专业维度**: 涵盖交叉学科/未来产业、传统升级/智能形态等27个种子专业
- **大学维度**: 双一流/C9 → 普通本科 → 应用型本科 → 高职专科 四层分层
- **城市维度**: 超一线 → 一线 → 新一线 → 二/三/四线，打破纯GDP论
- **产业维度**: 5大产业纲(人工智能、未来产业、先进制造、新能源、现代农业) + 10个细分目

## 快速开始

直接在浏览器中打开 `index.html` 即可使用，无需任何构建工具或后端服务。

或使用任意静态服务器：
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

## 数据扩展

修改 `js/data.js` 中的四个核心数组即可扩展数据：
- `MAJORS[]` — 专业数据
- `UNIVERSITIES[]` — 大学数据
- `CITIES[]` — 城市数据
- `INDUSTRY_TREE[]` — 产业纲目树

每个实体通过ID进行交叉引用，支持多对多关系。未来可替换为后端API。
